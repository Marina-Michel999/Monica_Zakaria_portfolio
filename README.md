# Portfolio — Monica Zakaria (Digital Marketing)

## تشغيل المشروع
```bash
npm install
npm start
```
هيفتحلك سيرفر محلي (عبر `serve`) وتقدري تفتحي الرابط في المتصفح.
لو مش عندك `serve` هيتظبط تلقائي أول مرة (npx بيحمله).

## فين تحطي الصور والفايلات بتاعتك
كل قسم في الـ Portfolio ليه فولدر مخصص جوه `assets/projects/`. المطلوب بس إنك تحطي الملفات بنفس الاسم المرقم والامتداد المكتوب، أو تغيّري الامتداد في ملف `js/main.js` (المتغير `portfolioCategories` في الأول) لو الامتداد مختلف.

| القسم | العدد | المكان | الامتداد الحالي |
|---|---|---|---|
| Graphics | 12 صورة | `assets/projects/graphics/1.jpg` لحد `12.jpg` | jpg |
| Freelancing | 2 صورة | `assets/projects/freelancing/1.jpg`, `2.jpg` | jpg |
| SEO | 3 ملفات | `assets/projects/seo/1.pdf`, `2.pdf`, `3.pdf` | pdf (غيّريه لو مش pdf) |
| Social Media | 14 صورة | `assets/projects/social/1.jpg` لحد `14.jpg` | jpg |
| Content Plan | ملف واحد | `assets/projects/content-plan/1.xlsx` | xlsx (غيّريه لو الملف Google Sheet مُصدَّر أو PDF مثلاً) |
| Web | صورة واحدة | `assets/projects/web/1.jpg` | jpg |

**صورتك الشخصية**: حطيها في `assets/profile.jpg` (نفس الاسم بالظبط) وهتظهر جوه الإطار الدائري بالتدرج اللوني تلقائيًا.

لحد ما تحطي الملفات، هيظهر بدالها placeholder ملوّن باسم القسم والرقم — عشان تعرفي مكان كل حاجة بالظبط.

## اللي اتعمل حسب طلبك
- الباكدجات (Bootstrap, Font Awesome) بقت عبر npm (`package.json`) مش لينكات يدوية.
- شيلت مكتبة AOS (animate on scroll) خالص لإنها كانت بتسبب اختفاء المحتوى لو حصل أي مشكلة في تحميلها. بدالها فيه أنيميشن بسيط بالـ CSS بس (fade-in للـ Hero) مش معتمد على أي مكتبة خارجية، فمش ممكن "يعلق" ويخفي حاجة.
- شيلت قسم Services خالص.
- اللون الرئيسي بقى تدرج (gradient) بنفسجي - فوشيا - ذهبي بدل اللون الأزرق.
- شكل الإطار حوالين الصورة اتغير لحلقة متدرجة (زي ستوري الانستجرام) بدل الشكل العضوي (blob) اللي كان موجود.
- فيه زرار Light / Dark mode في الـ navbar (بيحفظ اختيارك في المتصفح).
- قسم Portfolio فيه فلتر (زي navbar صغير) بالفئات الست، وكل صورة بتتكبر (lightbox) لما تدوسي عليها، والملفات (SEO / Content Plan) بتتفتح في تاب جديد.
- ضفت قسم "Experience & Education" من الـ CV بتاعك (مش موجود في التمبلت الأصلي) لإنه بيقوّي الصفحة، وممكن تشيليه لو مش عايزاه.

## لو عايزة تغيري التدرج اللوني
كل الألوان متجمعة في أول ملف `css/main.css` تحت `:root` — غيّري قيم `--grad`, `--accent-1/2/3` وهتتغير في كل الموقع تلقائي.
